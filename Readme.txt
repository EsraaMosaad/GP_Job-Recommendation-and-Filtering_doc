

#Job Recommendation And Filtering System

This repository contains the code for our project, where we trained different models on our dataset and extracted skills using the `TESAT_ALL_MODEL.ipynb` notebook. We also prepared a new dataset `new_data.csv` for training.

## Documentation

You can find the documentation for this project in the file named `name of doc`.

## Model Selection

After training different models, we found that the TF-IDF model provided the best results. Therefore, we used it for our deployment.

## Deployment

To deploy the project, you can use the code provided in the `app.file` file.
